## 一、实验名称

**Python 常用数据类型**

## 二、实验目的

熟悉 Python 的输入输出、两种字符串格式化方法、列表、元组、字典及集合的基本运用，重点熟悉列表及字符串的各种操作。

## 三、问题描述

### 1. 列表操作

#### 题目 1：利用集合去重

给定一个列表，其中有数据元素是重复的，请利用集合 `set` 的元素不能重复特性，去除重复值。

#### 题目 2：利用字典去重

给定一个列表，其中有数据元素是重复的，请利用字典的键不能重复特性，去除重复值。

### 2. 字符串操作

#### 题目 1：判断对称字符串

请编写一个程序，从键盘输入一个字符串，判断该字符串是否为对称字符串。

#### 题目 2：统计单词数量

编写一个程序，从键盘循环输入字符串，统计输入的各个单词的数量，并按照从大到小的次序打印输出各个单词的数量。

## 四、基本要求

1. 使用 Python 编写程序，完成上述列表操作和字符串操作题目。
2. 涉及输出的内容，需要使用 `format` 字符串格式化输出方法。
3. 程序应能够从键盘接收输入，并输出符合题目要求的结果。
4. 对于列表去重题，应分别使用 `set` 和字典键唯一性的方式实现。
5. 对于字符串操作题，应完成对称字符串判断和单词数量统计功能。
